Grasp is a lightweight Python toolkit for data mining (search engines, servers, HTML DOM + CSS selectors, plaintext), Natural Language Processing (NLP; tokenization, multilingual part-of-speech tagging & sentiment analysis), Machine Learning (ML; clustering, classification, confusion matrix, n-grams),

Grasp is a Swiss Army Knife of free, fast, short, clean AI algorithms, with ~25 lines of code per function and ~75 lines of code per class, with no external dependencies, with self-explanatory function names and parameters, bundled into one well-documented file: grasp.py.

def tokenize(s, language='en', separator=' '): # de|en|es|fr|nl

Returns string s with punctuation marks split from words, and sentences separated by newlines.

Regular expressions identify abbreviations (e.g., etc.), contractions (it's), numbers (1.0), symbols (:-o), URLs (www.), HTML tags (<html>), special characters (&amp;), and so on, with language-specific rules for de, en, es, fr, nl. An in-sentence separator or an end-of-sentence \n is then inserted at other punctuation marks.

Tokenization is a common preprocessing step for part-of-speech tagging and word count (i.e., so that Hi = 2x instead of Hi = 1x and Hi! = 1x), and in turn used for vectorization in Machine Learning algorithms.
